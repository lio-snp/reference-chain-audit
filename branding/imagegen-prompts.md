# Imagegen Prompts

These prompts are prepared for the `reference-chain-audit` project and follow the repository's current positioning: skill-first, agent-driven, evidence-chain focused.

## 1. Project Diagram

```text
Use case: infographic-diagram
Asset type: GitHub repository overview graphic
Primary request: create a clean architecture diagram for the reference-chain-audit project
Scene/background: minimal light background suitable for README usage
Subject: an agent-first audit workflow showing inputs, evidence-chain reasoning, optional helper scripts, and final decisions
Style/medium: polished technical infographic, vector-like, editorial, open-source project visual
Composition/framing: horizontal layout, clear blocks and arrows, strong hierarchy, spacious margins
Lighting/mood: crisp, calm, precise, trustworthy
Color palette: slate blue, deep teal, soft off-white, muted gold accents
Text (verbatim): "Inputs" "reference-chain-audit skill" "Existence" "Local Claim Relevance" "Landing Correctness" "Decision Layer" "Audit Report" "Style Follow-Up" "Optional Helpers"
Constraints: legible text, consistent alignment, repository-grade documentation graphic, no watermark
Avoid: stock-photo vibe, glossy 3D, purple neon, clutter, decorative icons that distract from the flow
```
